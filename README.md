# Resume Placeholder

This is a placeholder file for the resume PDF.

To add your actual resume:
1. Save your resume as `Mohammed_Hasan_Ali_Resume.pdf`
2. Place it in the `assets` folder
3. The download button in the navigation will automatically link to it

The resume should be professionally formatted and include:
- Contact information
- Professional summary
- Education details
- Work experience
- Projects
- Skills
- Certifications

Make sure the PDF is optimized for web viewing and not too large in file size.
